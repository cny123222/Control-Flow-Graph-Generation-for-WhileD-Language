# main.py 测试用例 Mermaid 流程图

本目录包含 `main.py` 中所有测试用例的 Mermaid 流程图。

## 如何查看

1. **在线查看**: 访问 https://mermaid.live/
2. **复制代码**: 将 MD 文件中的 Mermaid 代码块复制到编辑器
3. **自动渲染**: 右侧会自动显示流程图
